Key Features of HTML5

  • Responsive Design: Websites automatically adjust their layout and content based on the screen size.
  • Viewport Meta Tag: Ensures that websites scale correctly on different screens.
  • Flexible Multimedia: HTML5 supports native multimedia (<video> and <audio>) without requiring third-party plugins like Flash.
  • Cross-Platform Compatibility: Features such as the <canvas> element allow developers to build interactive and graphical content that works across multiple devices.
  • Forms and Input Types: Introduces new input types (email, tel, date) that adjust for different devices. For example, a tel input type brings up a numeric keypad on smartphones.
  • Offline Support: Technologies like localStorage and service workers enable offline capabilities so web apps work even without an internet connection.

Semantic Elements in HTML

Semantic elements clearly describe their meaning both to the browser and the developer. Examples include <header>, <footer>, <article>, <section>, and <nav>.

The <article> Element

The <article> Element represents a self-contained composition in a document, such as a blog post or news article.

Example:

<article>
  <h2>Latest News</h2>
  <p>This is a news article.</p>
</article>
