Lecture 6 – HTML Forms

The <form> Element

The <form> Element is the container for all form elements. It defines a form for collecting user input.

Example:

<form action="/submit" method="post">
  <!-- Form content here -->
</form>

Form Structure

Forms contain various elements for user interaction. Input fields, labels, buttons, and selection lists are all common form components.

Example:

<form action="/submit" method="post">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name">
</form>

The <button> Element

The <button> Element is used to submit the form or trigger JavaScript actions. Types include submit, reset, and button.

Example:

<form>
  <button type="submit">Submit</button>
</form>
