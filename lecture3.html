What is HTML?

Link → Hypertext Link

Hyper means beyond (outside, elsewhere)

Web Page → Hyper Text Document

Web Page Markup Language → Hyper Text Markup Language (HTML)

Markup Language: A markup language is a computer language that uses tags to define elements within a document.

HTML stands for HyperText Markup Language

It is the standard language used to create webpages

HTML defines the structure of a webpage using elements and tags

HTML vs XHTML

This table compares the key differences between HTML and XHTML.

HTML XHTML
HTML is an acronym for Hypertext Markup Language. XHTML is an acronym for Extensible Hypertext Markup Language.
It was created by Tim Berners-Lee in 1991. It was created by the W3C (World Wide Web Consortium) in 2000.
Tags and attributes do not have to be in lower or upper case. Every tag and attribute must be in lowercase.
It is not essential to write the DOCTYPE at the top. DOCTYPE declaration is mandatory at the top of the document.
Tags do not need to be closed in the same sequence they are opened. All tags must be properly nested and closed in sequence.
