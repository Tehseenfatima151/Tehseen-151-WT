Basic Table Structure

The basic structure of an HTML table includes table headers and data rows:

<table> 
  <thead>
    <tr>
      <th>Header 1</th> <th>Header 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Data 1</td> <td>Data 2</td>
    </tr>
  </tbody>
</table>

Introduction to Div

<div> is a block-level element used to group and structure content on a webpage.

  • Used for creating sections of a webpage.
  • Unlike tables, <div> is better suited for flexible layout design.

Basic Example of Div:

<div>
  <h1>Title</h1>
  <p>This is a paragraph inside a div.</p>
</div>
